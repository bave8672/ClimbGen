About this app

This tool generates a random training session for rock climbing.

Use the settings to select discipline(s) (lead, boulder, top rope) and any equipment (fingerboard, campusboard, weight belt). Use the sliders to select the focus of the exercises (power, endurance, or a combination of the two) and the session duration (from one exercise up to four).

Each exercise should take approximately 25-30 minutes.
